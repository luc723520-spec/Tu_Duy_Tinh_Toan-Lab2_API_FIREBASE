# Healthylifestyle Tracker - Tư Duy Tính Toán Lab 2

## Hướng dẫn cài đặt
1. Tạo môi trường ảo: `python -m venv venv`
2. Kích hoạt môi trường và cài đặt thư viện: `pip install -r requirements.txt`

## Hướng dẫn chạy
1. Backend: Bật terminal, chạy lệnh `uvicorn backend.main:app --reload`
2. Frontend: Bật một terminal khác, chạy lệnh `streamlit run frontend/app.py`
